# Contributing

Thank you for helping improve Learn Rust by Building.

## Course principles

- Introduce one new idea at a time.
- Ask learners to predict before they run code.
- Connect every lesson to the cumulative dispatch-desk project.
- Prefer clear standard-library code over clever abstractions.
- Link conceptual claims to the relevant Rust Book chapter.

## Making a change

1. Keep starter projects compiling unless the exercise explicitly studies a
   compiler error.
2. Keep reference solutions focused on concepts already introduced.
3. Add tests for new business rules and important boundaries.
4. Run `bash scripts/check.sh` from Git Bash.

Documentation examples should use clearly fictional, globally varied names and
locations. Do not include real personal or customer data, and do not copy prose
from the Rust Book.
