# Module 4 — Useful Collections

## Outcome

Manage several orders as a delivery queue and summarize operational data.

This module follows [Rust Book Chapter 8](https://doc.rust-lang.org/book/ch08-00-common-collections.html).

## Lesson sequence

| Lesson | Main idea | Checkpoint |
|---|---|---|
| [1. Vectors](01-vectors.md) | Ordered growable values | Store orders |
| [2. Iteration](02-iteration.md) | Borrow, mutate, or consume items | Find work |
| [3. Strings](03-strings.md) | UTF-8 text operations | Search customers |
| [4. Hash maps](04-hash-maps.md) | Keyed aggregation | Count statuses |
| [5. Queue operations](05-queue-operations.md) | Collection-backed policy | Dispatch priority first |
| [Practical](06-practical-delivery-queue.md) | Combine collections | Build a queue report |
